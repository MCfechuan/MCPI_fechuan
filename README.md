# MC_飞船个人网站

这是一个为哔哩哔哩UP主"MC_飞船"创建的个人介绍网站。MC_飞船是一位知名的Minecraft PVP玩家，同时精通Python、C++、Java等多种编程语言的技术创作者。

## 🎮 网站功能

- **响应式设计**：适配桌面、平板和手机设备
- **现代化UI**：使用CSS变量、Flexbox和Grid布局
- **交互效果**：平滑滚动、动画、悬停效果
- **Git部署**：完全支持GitHub Pages部署
- **SEO优化**：包含元标签、语义化HTML结构
- **性能优化**：懒加载、代码分割、缓存控制
